# Pokémon Explorer

A beginner-friendly **Python + Streamlit** project that uses the **PokéAPI** to explore Pokémon data.

## Features
- Search Pokémon by **name or ID**
- Display **images** (front, back, official artwork)
- Show **types** (with badges)
- Show **base stats** in **interactive charts**
- Display **evolution chain**
- "Surprise Me!" button for random Pokémon
